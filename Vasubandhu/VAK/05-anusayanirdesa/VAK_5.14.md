# VAK_5.14 — Six Anuśayas with Uncontaminated Objects

## 1. Sanskrit — Devanāgarī

```sanskrit
मिथ्यादृग्विमती ताभ्यां युक्ताविद्याथ केवला ।
निरोधमार्गदृग्घेयाः षडनास्रवगोचराः ॥ ५.१४ ॥
```

## 2. Sanskrit — IAST

```text
mithyādṛg-vimatī tābhyāṃ yuktāvidyātha kevalā |
nirodha-mārga-dṛg-gheyāḥ ṣaḍ anāsrava-gocarāḥ || 5.14 ||
```

The local source gives:

```text
mithyādṛgvimatī tābhyāṃ yuktāvidyātha kevalā /
nirodhamārgadṛggheyāḥ ṣaḍanāsravagocarāḥ // VAkK_5.14 //
```

The kārikā identifies six anuśayas abandoned through seeing cessation and the path, and marks them as taking uncontaminated determinations as their objects.

---

## 3. Padaccheda

```text
mithyā-dṛk |
vimatiḥ |
tābhyām |
yuktā avidyā |
atha |
kevalā |
nirodha-mārga-dṛk-gheyāḥ |
ṣaṭ |
```

The three relevant forms are:

```text
mithyā-dṛṣṭi
    false view

vimati
    doubt

avidyā
    ignorance
        associated with those two
        or independent
```

These occur with respect to two truth-fields:

```text
nirodha
    cessation

mārga
    path
```

Thus:

```text
3 anuśayas × 2 truth-fields = 6
```

---

## 4. Grammar

| Term | Form | Working meaning |
|---|---|---|
| `mithyā-dṛk` | nominative singular, contracted from `mithyā-dṛṣṭi` | false view |
| `vimatiḥ` | nominative feminine singular | doubt, divided judgment |
| `tābhyām` | instrumental dual | with those two |
| `yuktā avidyā` | nominative feminine singular | ignorance conjoined with them |
| `atha` | connective particle | and then, further |
| `kevalā` | nominative feminine singular | isolated, independent |
| `nirodha-mārga-dṛg-gheyāḥ` | nominative masculine plural compound | to be abandoned through seeing cessation and the path |
| `ṣaṭ` | numeral | six |
| `anāsrava-gocarāḥ` | nominative masculine plural | having uncontaminated objects |

The instrumental dual `tābhyām` refers back to:

```text
mithyā-dṛṣṭi
vimati
```

so `tābhyāṃ yuktā avidyā` is ignorance associated with false view and doubt.

---

## 5. Literal translation

> False view, doubt, ignorance conjoined with those two, and independent ignorance—those abandoned through seeing cessation and the path are six, and they have uncontaminated objects.

---

## 6. Philosophical translation

> With respect to cessation and the path, the afflictive forms are false view, doubt, ignorance associated with those two, and independent ignorance. Distributed across these two truth-fields, they make six. Their peculiarity is that, although they are themselves contaminated determinations, the objects they misconceive are uncontaminated: cessation and the path.

The structure is:

```text
contaminated cognition
    can take
uncontaminated determination
    as its object
```

This is a crucial refinement. The status of a cognition and the status of its object must not be conflated.

---

# 7. The sixfold count

For cessation:

```text
1. false view concerning cessation
2. doubt concerning cessation
3. ignorance concerning cessation
```

For the path:

```text
4. false view concerning the path
5. doubt concerning the path
6. ignorance concerning the path
```

The verse compresses associated and independent ignorance into the category `avidyā`, but distinguishes their modes of occurrence.

```text
ignorance with false view or doubt
ignorance occurring independently
```

The Bhāṣya must clarify whether these are counted separately in every context or only distinguished functionally within each of the two truth-fields.

---

# 8. `Anāsrava-gocara`

```text
an-āsrava
    without outflow
    uncontaminated

gocara
    range
    field
    object-domain
```

The kārikā states that these anuśayas have uncontaminated objects.

That is:

```text
nirodha
mārga
```

are not themselves contaminated merely because contaminated cognition takes them as objects.

This yields a strict relation:

```text
object-status
    ≠
cognition-status
```

A false cognition does not contaminate its object ontologically.

---

# 9. False relation to a true object

The verse therefore introduces a subtle form of error.

Earlier verses often concerned false objects or false predicates imposed upon conditioned existence. Here the object itself is path-valid and uncontaminated, while the cognitive relation to it is defective.

```text
true or liberative object
    +
false relation
    =
afflicted cognition
```

Examples of the general structure are:

```text
cessation is denied
cessation is doubted
cessation is not understood

the path is denied
the path is doubted
the path is not understood
```

The error lies in the determination of the object, not in the object's own nature.

---

# 10. Positive dialectic

The positive dialectical movement becomes very clear:

```text
false view of cessation
    → seeing cessation

doubt concerning cessation
    → determination through seeing cessation

ignorance concerning cessation
    → cognition of cessation

false view of the path
    → seeing the path

doubt concerning the path
    → determination through seeing the path

ignorance concerning the path
    → cognition of the path
```

The corrective operation is specified by the very truth that had been misconceived.

```text
error is not removed abstractly

it is removed
    through adequate relation
    to the same determinate field
```

This is not merely negation of error. It is the production of the adequate positive relation.

---

# 11. Cessation and path as distinct objects

The verse keeps `nirodha` and `mārga` separate.

```text
nirodha
    the accomplished absence or cessation of affliction

mārga
    the effective course by which that cessation is realized
```

Thus two different failures are possible:

```text
failure to recognize the end
failure to recognize the means
```

A cognition may accept cessation while misconceiving the path, or speak of a path while denying genuine cessation.

The scientific classification therefore requires both truth-fields.

---

# 12. Relation to VAK_5.12–5.13

VAK_5.12–5.13 analyzed universal anuśayas operating through the conditioned field.

VAK_5.14 introduces a contrast:

```text
universal anuśayas
    range widely through contaminated determinations

six special anuśayas
    take uncontaminated cessation and path as objects
```

The chapter is now classifying anuśayas not only by species and mode of abandonment, but by object-domain:

```text
contaminated object
uncontaminated object
```

This prepares the technical explanation of the specific range of cessation and path in the next verses.

---

# 13. Restrained Organon interpretation

The verse supplies a valuable logical distinction:

```text
invalid operation
    may target
valid content
```

Therefore a system cannot infer the validity of a cognitive act merely from the value of its target.

```text
uncontaminated object
    does not guarantee
uncontaminated cognition
```

For reflective logic:

```text
Object validity
Relation validity
Judgment validity
```

must be represented separately.

The fault may belong to the relation or judgment even where the object is sound.

# 15. Bhāṣya openings

The Bhāṣya pass should recover:

```text
1. why only false view, doubt, and ignorance occur here
2. the exact relation between associated and independent ignorance
3. why cessation and path are called uncontaminated objects
4. how an afflicted cognition takes an uncontaminated object
5. whether all six are counted symmetrically across the two truths
6. the precise force of nirodha-mārga-dṛg-gheya
```

The kārikā securely establishes:

```text
six anuśayas
    abandoned through seeing cessation and path
    having uncontaminated objects
```

It does not yet expose the full doctrinal machinery that supports the classification.

---

# 16. Working conclusion

```text
contaminated cognition
    can oppose
uncontaminated cessation

contaminated cognition
    can oppose
uncontaminated path
```

Therefore liberation requires more than presenting a valid object.

```text
valid object
    must be met by
adequate seeing
```

The next verse specifies the levels to which cessation and the path belong and how these objects are distributed across the spheres.
