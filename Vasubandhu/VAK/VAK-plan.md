# VAK Plan

## Project Method

The work proceeds in three passes.

```text
Pass 1 — Kārikā Pass
    determinate structural spine
    Sanskrit-first verse files
    light doctrinal and logical determination
    commit one verse at a time

Pass 2 — Bhāṣya Pass
    fair analysis of Vasubandhu's reasoning
    vocabulary of reflection
    logo-genesis / reflected paths of dharmas
    identify objections, definitions, grounds, and doctrinal controls

Pass 3 — Our Comment
    Organon interpretation
    positive synthetic commentary
    path-discrimination framework
    remain true to Vasubandhu's text while shifting into our model
```

Hegel-clean orientation:

```text
Kārikā Pass
    = Being / determinate structural field

Bhāṣya Pass
    = Essence / Reflection / dialectical-negative mediation

Our Comment
    = Concept / positive synthetic Organon interpretation
```

## Current Principle

The Kośa presents the whole Rational–Empirical Field so that the field can be discriminated as:

```text
Path / not-Path
kliṣṭa / akliṣṭa
avidyā / vidyā
```

The first pass is not final commentary. It builds the field by walking it.
