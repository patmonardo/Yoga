# VAK_2.53 — Prayogaja Profiles and Saṃprayuktaka-Hetu

## 1. Sanskrit — Devanāgarī

```text
प्रयोगजास्तयोरेव श्रुतचिन्तामयादिकाः ।
संप्रयुक्तकहेतुस्तु चित्तचैत्ताः समाश्रयाः ॥ VAkK_2.53 ॥
```

## 2. Sanskrit — IAST

```text
prayogajās tayor eva śrutacintāmayādikāḥ /
saṃprayuktakahetus tu cittacaittāḥ samāśrayāḥ // VAkK_2.53 //
```

## 3. Literal Translation

```text
Those born of application, such as those made of hearing and reflection, belong to those same cases.
But the associated ground is citta and caittas having the same support.
```

## 4. Philosophical Translation

```text
Cultivated states born of application — beginning with hearing and reflection — follow the same rule.
The associated ground is defined precisely: citta and caittas sharing one support.
```

## 6. Logical Determination

This verse confirms our `2.34` reading.

```text
Saṃprayuktaka-Hetu:
    citta + caittas
    sharing the same support
    function as associated ground.
```

Thus citta-caitta association is not just phenomenology; it becomes an objective ground-type.
