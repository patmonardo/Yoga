# VAK_2.52 — Sabhāga-Hetu: Homogeneous Ground

## 1. Sanskrit — Devanāgarī

```text
सभागहेतुः सदृशाः स्वनिकायभुवः अग्रजाः ।
अन्योऽन्यं नवभूमिस्तु मार्गः समविशिष्टयोः ॥ VAkK_2.52 ॥
```

## 2. Sanskrit — IAST

```text
sabhāgahetuḥ sadṛśāḥ svanikāyabhuvaḥ agrajāḥ /
anyo 'nyaṃ navabhūmis tu mārgaḥ samaviśiṣṭayoḥ // VAkK_2.52 //
```

## 3. Literal Translation

```text
The homogeneous ground consists of similar factors, arisen in their own class and level, prior-born.
The path, however, in the nine levels, is mutual for equal and superior cases.
```

## 4. Philosophical Translation

```text
Sabhāga-hetu is the ground of similarity: like gives rise to like within its own class, level, and prior sequence.
In the case of the path, the rule is expanded across the nine levels, allowing equal and superior relations.
```

## 5. Technical Vocabulary

| Sanskrit | Working Translation | Note |
|---|---|---|
| sabhāga-hetu | homogeneous ground | Similar-class cause-ground |
| sadṛśa | similar | Like-to-like relation |
| svanikāya | own class / own group | Class-domain |
| bhū | level / ground | Realm/stage-level |
| agraja | prior-born | Temporal priority |

| samaviśiṣṭa | equal and superior | Path relation across equal/superior levels |

## 6. Logical Determination

```text
if X is similar to Y,
and X belongs to the same class/level,
and X is prior-born,
then X may be sabhāga-hetu for Y.
```

This is objective repetition as ground:

```text
similarity + class + level + priority = homogeneous ground
```
