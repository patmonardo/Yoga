# VAK_2.50 — Kāraṇa-Hetu and Sahabhū-Hetu

## 1. Sanskrit — Devanāgarī

```text
स्वतोऽन्ये कारणं हेतुः सहभूर्ये मिथःफलाः ।
भूतवच्चित्तचित्तानुवर्तिलक्षणलक्ष्यवत् ॥ VAkK_2.50 ॥
```

## 2. Sanskrit — IAST

```text
svato 'nye kāraṇaṃ hetuḥ sahabhūr ye mithaḥphalāḥ /
bhūtavac cittacittānuvartilakṣaṇalakṣyavat // VAkK_2.50 //
```

## 3. Literal Translation

```text
Other than itself is the kāraṇa-hetu.
Those that are mutually fruits are sahabhū,
like the great elements, and like citta with its followers, mark and marked.
```

## 4. Philosophical Translation

```text
The conditioning ground is whatever other than the item functions as its enabling condition.
The co-arisen ground is a mutual ground-result relation: factors arise together and support one another,
as with the great elements, citta and its concomitants, and mark with marked item.
```

## 5. Technical Vocabulary

| Sanskrit | Working Translation | Note |
|---|---|---|
| svataḥ anye | other than itself | Defines kāraṇa broadly |
| kāraṇa-hetu | conditioning ground | General enabling ground |
| sahabhū | co-arisen | Simultaneously arising/mutually supporting |
| mithaḥ-phala | mutual fruit | Each functions as result for the other |
| bhūta | great element | Example of mutual co-grounding |
| citta-cittānuvartin | citta and followers | Citta and citta-following factors |
| lakṣaṇa-lakṣya | mark and marked | Mark and what is marked |

## 6. Logical Determination

```text
Kāraṇa-Hetu:
    if X is other than Y and enables Y,
    X may function as conditioning ground of Y.

Sahabhū-Hetu:
    if X and Y are mutually fruits,
    X and Y are co-arisen grounds.
```

This is harsh Objective Logic: the event is placed in a network of enabling and mutual co-arising facts.

## 7. Organon Interpretation

```text
ConditioningEdge:
    source != target
    relation = kāraṇa

CoArisingEdge:
    source <-> target
    relation = mutualFruit
```
