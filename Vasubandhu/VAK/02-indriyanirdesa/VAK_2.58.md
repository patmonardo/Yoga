# VAK_2.58 — Puruṣakāraja and Adhipati Results

## 1. Sanskrit — Devanāgarī

```text
यद्वलाज्जायते यत्तत्फलं पुरुषकारजम् ।
अपूर्वः संस्कृतस्यैव संस्कृतोऽधिपतेः फलम् ॥ VAkK_2.58 ॥
```

## 2. Sanskrit — IAST

```text
yadvalāj jāyate yat tat phalaṃ puruṣakārajam /
apūrvaḥ saṃskṛtasyaiva saṃskṛto 'dhipateḥ phalam // VAkK_2.58 //
```

## 3. Literal Translation

```text
Whatever arises by the force of whatever, that result is puruṣakāra-born.
What is unprecedented and conditioned is the conditioned result of the dominant factor.
```

## 4. Philosophical Translation

```text
A result produced through operative force is called puruṣakāraja.
The dominant result is a new conditioned outcome arising under the efficacy of a dominant condition.
```

## 5. Technical Vocabulary

| Sanskrit | Working Translation | Note |
|---|---|---|
| yad-valāt | by the force of which | Operative power condition |
| jāyate | arises | Arising-event |
| puruṣakāraja | produced by effort/operation | Result born from operative activity |
| apūrva | unprecedented / new | Not previously arisen in that way |
| saṃskṛta | conditioned | Produced formation |
| adhipati | dominant / governing | Dominant efficacy |
| phala | result | Effect/result type |

## 6. Logical Determination

```text
PuruṣakārajaPhala:
    if result arises by force of operation,
    result = puruṣakāraja.

AdhipatiPhala:
    if a new conditioned result arises under dominance,
    result = adhipati-phala.
```

This preserves the objective fact: result follows operative force, not private intention.

## 7. Organon Interpretation

```text
OperationalResult:
    sourceForce -> result

DominantResult:
    dominantCondition -> newConditionedOutcome
```

This is transformation semantics for the Dataset.
