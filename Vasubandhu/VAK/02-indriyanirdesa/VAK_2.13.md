# VAK_2.13 — Abandonment Classification of the Faculties

## 1. Sanskrit — Devanāgarī

```text
मनोवित्तित्रयं त्रेधा द्विहेया दुर्मनस्कता ।
नव भावनया पञ्च त्वहेयान्यपि न त्रयम् ॥ VAkK_2.13 ॥
```

## 2. Sanskrit — IAST

```text
manovittitrayaṃ tredhā dviheyā durmanaskatā /
nava bhāvanayā pañca tv aheyāny api na trayam // VAkK_2.13 //
```

Source form:

```text
manovittitrayaṃ tredhā dviheyā durmanaskatā /
nava bhavanayā pañca tvaheyānyapi na trayam // VAkK_2.13 //
```

## 3. Textual Caution

This verse classifies faculties by their relation to abandonment. The terminology is compressed and count-sensitive. Final set-membership should be checked later against the Bhāṣya.

## 4. Literal Translation

```text
Mind and the triad of feelings are threefold.
Distress is to be abandoned in two ways.
Nine are to be abandoned by cultivation.
Five, however, are not to be abandoned; also not the triad.
```

## 5. Working Interpretation

The verse introduces the abandonment axis.

```text
heya
    = to be abandoned

dvi-heya
    = to be abandoned in two ways

bhāvanayā heya
    = to be abandoned by cultivation

aheya
    = not to be abandoned
```

This shifts the classification from what realm a faculty belongs to, to how it is transformed or left behind on the path.

## 6. Technical Vocabulary

| Sanskrit | Working Translation | Note |
|---|---|---|
| mano | mind-faculty | Classified threefold here |
| vitti-traya | triad of feelings | Compressed term; probably a counted group of feeling faculties |
| tredhā | threefold | Three-mode classification |
| dvi-heya | to be abandoned in two ways | Technical abandonment class |
| durmanaskatā | distress / bad-mindedness | Related to daurmanasya |
| bhāvanā | cultivation | Path of cultivation/development |
| bhāvanayā heya | abandonable by cultivation | Removed through development |
| aheya | not abandonable / not to be abandoned | Not rejected in the path-classification |

## 7. Logical Determination

The faculties are not only classified by realm and purity; they are classified by their path-transformation.

```text
faculty
    = governing efficacy
    + realm-status
    + purity-status
    + abandonment-status
```

The path does not simply erase the faculty-system. It differentiates which faculties are abandoned, transformed, retained, or purified.

## 8. Organon Seed

Organon reading:

```text
abandonment-status
    = path-operation applied to the faculty-driver

bhāvanayā heya
    = removed by cultivation-process

aheya
    = retained / not rejected by the path
```

The faculty-system is now treated as an object of discipline.

## 9. Compact Formula

```text
VAK_2.13 classifies faculties by abandonment-status:
some are threefold, distress is twofoldly abandonable,
nine are abandoned by cultivation, and five plus the triad are not abandoned.
```
