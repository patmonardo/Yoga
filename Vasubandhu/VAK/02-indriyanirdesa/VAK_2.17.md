# VAK_2.17 — Arhatship and Minimum Faculty-Configuration

## 1. Sanskrit — Devanāgarī

```text
एकादशभिरर्हत्त्वमुक्तं त्वेकस्य संभवात् ।
उपेक्षजीवितमनोयुक्तोऽवश्यं त्रयान्वितः ॥ VAkK_2.17 ॥
```

## 2. Sanskrit — IAST

```text
ekādaśabhir arhattvam uktaṃ tv ekasya saṃbhavāt /
upekṣajīvitamanoyukto 'vaśyaṃ trayānvitaḥ // VAkK_2.17 //
```

Source form:

```text
ekādaśabhirarhattvamuktaṃ tvekasya saṃbhavāt /
```

## 3. Textual Caution

This verse begins a count analysis of which faculties are present in a given configuration. The exact scenarios should be checked later against the Bhāṣya.

## 4. Literal Translation

```text
Arhatship is said to be with eleven, because one is possible.
A configuration with equanimity, life, and mind is necessarily accompanied by three.
```

## 5. Working Interpretation

The verse introduces faculty-configuration analysis.

```text
anvita / yukta
    = accompanied by / configured with
```

Two cases are marked:

```text
arhatship
    -> counted with eleven faculties

minimal necessary configuration
    -> upekṣā + jīvita + manas
       three faculties necessarily present
```

The second half is especially important for the speculative seed: the minimal operating profile of a living cognitive stream is neutral feeling, life, and mind.
