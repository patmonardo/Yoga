# VAK_2.56 — Hetu to Phala Mapping

## 1. Sanskrit — Devanāgarī

```text
विपाकफलमन्त्यस्य पूर्वस्याधिपतं फलम् ।
सभाग सर्वत्रगयोर्निष्यन्दः पौरुषं द्वयोः ॥ VAkK_2.56 ॥
```

## 2. Sanskrit — IAST

```text
vipākaphalam antyasya pūrvasyādhipataṃ phalam /
sabhāga sarvatragayor niṣyandaḥ pauruṣaṃ dvayoḥ // VAkK_2.56 //
```

## 3. Literal Translation

```text
The last has maturation-fruit; the first has dominant fruit.
For sabhāga and sarvatraga there is niṣyanda; for two there is pauruṣa fruit.
```

## 4. Philosophical Translation

```text
Each ground-type maps to a specific result-type.
The maturation ground yields maturation-fruit; the conditioning ground yields dominant fruit;
homogeneous and pervasive grounds yield homogeneous outflow; and two grounds yield effort-produced result.
```

## 6. Logical Determination

VAK_2.56 gives the mapping table:

```text
vipāka-hetu       -> vipāka-phala
kāraṇa-hetu       -> adhipati-phala
sabhāga-hetu      -> niṣyanda-phala
sarvatraga-hetu   -> niṣyanda-phala
sahabhū-hetu      -> pauruṣa-phala
saṃprayuktaka-hetu -> pauruṣa-phala
```

This is the Hetu–Phala engine proper.
