# VAK_2.54 — Sarvatraga-Hetu and Vipāka-Hetu

## 1. Sanskrit — Devanāgarī

```text
सर्वत्रगाख्यः क्लिष्टानां स्वभूमौ पूर्वसर्वगाः ।
विपाकहेतुरशुभाः कुशलाश्चैव सास्रवाः ॥ VAkK_2.54 ॥
```

## 2. Sanskrit — IAST

```text
sarvatragākhyaḥ kliṣṭānāṃ svabhūmau pūrvasarvagāḥ /
vipākahetur aśubhāḥ kuśalāś caiva sāsravāḥ // VAkK_2.54 //
```

## 3. Literal Translation

```text
That called sarvatraga is of the afflicted: prior, all-going within its own level.
The maturation ground consists of unwholesome factors and wholesome factors with outflows.
```

## 4. Philosophical Translation

```text
Sarvatraga-hetu is the pervasive afflicted ground: prior afflicted factors range throughout their own level.
Vipāka-hetu is the ground of maturation, produced by unwholesome and outflow-bound wholesome factors.
```

## 5. Technical Vocabulary

| Sanskrit | Working Translation | Note |
|---|---|---|
| sarvatraga | omnipresent / all-going | Pervasive ground |
| kliṣṭa | afflicted | Defiled or disturbed status |

| pūrva | prior | Temporal priority |
| vipāka-hetu | maturation ground | Ground of maturation-result |
| aśubha | unwholesome | Unskillful/negative status |
| kuśala sāsrava | wholesome with outflows | Conditioned wholesome still bound by outflow |

## 6. Logical Determination

```text
Sarvatraga-Hetu:
    prior afflicted factors
    within their own level
    function pervasively.

Vipāka-Hetu:
    unwholesome factors
    and outflow-bound wholesome factors
    function as maturation grounds.
```

This is objective karmic facticity: ground-types determine later maturation without appeal to subjective wish.
