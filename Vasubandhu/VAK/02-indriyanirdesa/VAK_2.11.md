# VAK_2.11 — Wholesome, Twofold, Threefold, and Onefold Faculties

## 1. Sanskrit — Devanāgarī

```text
मनोऽन्यवित्तिश्रद्धादीन्यष्टकं कुशलं द्विधा ।
दौर्मनस्यं मनोऽन्या च वित्तिस्त्रेधा अन्यदेकधा ॥ VAkK_2.11 ॥
```

## 2. Sanskrit — IAST

```text
mano 'nyavittiśraddhādīny aṣṭakaṃ kuśalaṃ dvidhā /
daurmanasyaṃ mano 'nyā ca vittis tredhā anyad ekadhā // VAkK_2.11 //
```

Source form:

```text
mano 'nyavittiśraddhādīni aṣṭakaṃ kuśalaṃ dvidhā /
daurmanasyaṃ mano 'nyā ca vittistredhā anyadekadhā // VAkK_2.11 //
```

## 3. Textual Caution

This is a dense classificatory verse. It belongs to the Abhidharma sorting of the faculties by ethical quality and modal status. The exact count-sets should be checked against the Bhāṣya later. This file preserves the kārikā as speculative seed.

## 4. Literal Translation

```text
Mind, the other feelings, and those beginning with faith — this eightfold set is wholesome and twofold.
Distress, mind, and the other feeling are threefold; the rest is onefold.
```

## 5. Working Interpretation

VAK_2.11 continues the classification of the faculties, now by ethical and modal status.

```text
kuśala
    = wholesome / constructive

dvidhā
    = twofold

tredhā
    = threefold

ekadhā
    = onefold
```

The verse marks that some faculties are determinately wholesome, some can occur in multiple modes, and some have only one relevant classification.

## 7. Logical Determination

The verse adds an ethical-modal layer to the indriya system.

```text
faculty
    = governing efficacy
    + path-status
    + purity-status
    + result-status
    + ethical/modal status
```

The point is not merely that faculties exist. They operate under determinate qualitative modes.

## 8. Organon Seed

Organon reading:

```text
kuśala-indriya
    = faculty-driver operating constructively for the path

tredhā-indriya
    = faculty-driver capable of three modal determinations

ekadhā-indriya
    = faculty-driver with one fixed mode in this classification
```

This is a classification of the *mode of operation* of the faculty-system.
