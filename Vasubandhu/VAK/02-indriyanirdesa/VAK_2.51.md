# VAK_2.51 — Citta-Followers and Shared Profiles

## 1. Sanskrit — Devanāgarī

```text
चैत्ता द्वौ संवरौ तेषां चेतसो लक्षणानि च ।
चित्तानुवर्तिनः कालफलादिशुभतादिभिः ॥ VAkK_2.51 ॥
```

## 2. Sanskrit — IAST

```text
caittā dvau saṃvarau teṣāṃ cetaso lakṣaṇāni ca /
cittānuvartinaḥ kālaphalādiśubhatādibhiḥ // VAkK_2.51 //
```

## 3. Literal Translation

```text
The caittas, the two restraints, and the marks of citta are citta-followers,
through time, fruit, and so forth, including wholesomeness and the like.
```

## 4. Philosophical Translation

```text
Certain factors follow citta in shared profiles: mental factors, restraints, and the marks of citta.
They are coordinated with citta through time, result, ethical status, and other shared determinations.
```

## 6. Logical Determination

VAK_2.51 defines a followership rule:

```text
if a factor is cittānuvartin,
then it shares citta's time, result, and status profiles.
```

This extends the citta-caitta association into the Hetu engine.
