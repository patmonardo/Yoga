# VAK_3.73 — World-System Scale: The Small Thousandfold World-System

## 3. Source Caution

The source reading is:

```iast
caturdvīpakacandrārkamerukāmadivaukasām /
brahmalokasahasraṃ ca sāhasraścūḍiko mataḥ // VAkK_3.73 //
```

Sandhi has been lightly separated:

```iast
catur-dvīpaka-candra-arka-meru-kāma-divaukasām /
brahma-loka-sahasram ca sāhasraḥ cūḍikaḥ mataḥ //
```

`cūḍika` is provisionally treated as the “small” thousandfold world-system, the first-level thousandfold aggregation. The exact cosmological term should be checked against the bhāṣya and printed editions.

## 6. Literal Translation

A thousand of four-continent worlds, moons, suns, Merus, desire-realm heavenly dwellers, and Brahma-worlds is considered the small thousandfold world-system.

## 7. Philosophical Translation

The cosmological field now expands from a single vertical world-order to a thousandfold aggregation: one thousand four-continent world-systems, with their moons, suns, Merus, desire-realm gods, and Brahma-worlds. This is called the small thousandfold world-system.

## 9. Doctrinal Determination

VAkK_3.72 established the visibility horizon of stations within the vertical deva/dhyāna field. VAkK_3.73 now expands the scale from one world-order to a thousandfold system.

```text
single world-order
    → four-continent unit
    → moon / sun / Meru / kāma-devas / Brahma-world
    → one thousand such units
    → small thousandfold world-system
```

This is still world-field, not liberation. The scale expands, but the Discriminator must still evaluate it as structured saṃsāric territory.

## 10. Logical Determination

This verse introduces iteration of the complete world-unit.

```text
world-unit
    × 1000
        = small thousandfold world-system
```

The Kośa now shifts from vertical hierarchy to replicated totality: the single world-order becomes a countable unit within a larger cosmological aggregate.

## 11. Interpretive Note

This is where the gods-section widens beyond a local ascent. Sensibility is now cosmic-scale: suns, moons, Merus, desire-heavens, and Brahma-worlds become repeatable units.

```text
refined territory
    → repeated world-system
    → enlarged field of appearance
```

The important path-discrimination remains: greater scale is not greater freedom. Expansion of world-field is still not release from world-field.

## 12. Organon Interpretation

Light contact point:

```text
The world is not merely elevated; it is iterable.
```

The observer-territory relation now becomes large-scale. A territory can be vertically stratified and horizontally multiplied. The Path-Discriminator must therefore distinguish not only gross from subtle, but local refinement from cosmological scale.
