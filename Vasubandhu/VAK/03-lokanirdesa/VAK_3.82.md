# VAK_3.82 — Time-Measure: Naraka Lifespans by Kāma-Deva Time

## 1. Sanskrit — Devanāgarī

```sanskrit
कामदेवायुषा तुल्या अहोरात्रा यथाक्रमम् ।
संजीवादिषु षट्स्वायुस्तैस्तेषां कामदेववत् ॥ VAkK_3.82 ॥
```

## 2. Sanskrit — IAST

```iast
kāmedevāyuṣā tulyā ahorātrā yathākramam /
saṃjīvādiṣu ṣaṭsv āyus tais teṣāṃ kāmadevavat // VAkK_3.82 //
```

## 3. Source Caution

The source reading is:

```iast
kāmedevāyuṣā tulyā ahorātrā yathākramam /
saṃjīvādiṣu ṣaṭsu āyustaisteṣāṃ kāmadevavat // VAkK_3.82 //
```

Sandhi has been lightly separated:

```iast
kāma-deva-āyuṣā tulyāḥ ahorātrāḥ yathā-kramam /
saṃjīva-ādiṣu ṣaṭsu āyuḥ taiḥ teṣām kāma-deva-vat //
```

The verse is read as establishing the lifespans of the first six hot narakas beginning with Saṃjīva. Their day-and-night units correspond, in order, to the lifespans of the kāma-devas; by those day-and-night units, their lifespans are measured like those of the kāma-devas. This should be checked against the bhāṣya.

## 6. Literal Translation

The day-and-nights are equal, in order, to the lifespans of the kāma-devas. In the six beginning with Saṃjīva, their lifespan is by those [day-and-nights], like the kāma-devas.

## 7. Philosophical Translation

The time-scale of the lower hot narakas is measured by an inverted divine standard. For the six narakas beginning with Saṃjīva, a single day-and-night corresponds successively to the lifespan of a kāma-deva class, and the naraka lifespan is then counted by those units as the kāma-deva lifespans are counted.

## 9. Doctrinal Determination

VAkK_3.79–3.80 measured kāma-deva lifespans through relative day-and-night scales. VAkK_3.81 raised the scale to kalpas and mahākalpas. VAkK_3.82 now applies realm-specific time-measure to naraka.

```text
kāma-deva lifespan
    → becomes day-and-night unit
        → for the six narakas beginning with Saṃjīva

naraka lifespan
    → counted by those day-and-night units
    → like the kāma-devas
```

This is a striking inversion: divine lifespan becomes the clock-unit of hellish duration.

## 10. Logical Determination

This verse introduces cross-realm temporal conversion by analogy.

```text
one realm's lifespan
    = another realm's day-and-night
```

The same principle seen in deva time is now extended downward. Loka is a system of relative clocks, not one homogeneous time-container.

## 11. Interpretive Note

This is a strong path-discrimination verse. The same cosmological machinery that measures divine enjoyment also measures hellish suffering. The point is not moral rhetoric alone; it is formal: both deva and naraka are time-regimes within saṃsāra.

```text
deva time
    → long enjoyment

naraka time
    → long suffering

both
    → measured duration
    → not liberation
```

The Discriminator is not impressed by height or terrified by depth; both are determinate world-times.

## 12. Organon Interpretation

Light contact point:

```text
Saṃsāric time is relational across realms.
```

For this pass, we observe the Kośa’s presentation: a lifespan in one realm can function as the day-and-night unit of another. The world-field is thus not only spatially and temporally stratified; its time-scales are mutually convertible by ordered correspondence.
