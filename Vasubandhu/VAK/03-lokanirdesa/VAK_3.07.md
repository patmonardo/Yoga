## 5. Syntax / Construction

The verse has two major operations.

First operation:

```iast
anicchāvasanān nānye
```

This explains why no further **vijñāna-sthiti** are counted beyond the seven named in VAkK 3.6.

Feature extraction:

```text
Negation:
    na anye
    no others

Reason / ground:
    anicchā-avasanāt
    because residence there is undesired
```

Meaning:

```text
There are no other [knowledge-stations], because residence there is not desired.
```

Second operation:

```iast
catasraḥ sthitayaḥ punaḥ /
catvāraḥ sāsravāḥ skandhāḥ svabhūmāv eva kevalam
```

This introduces another technical use of **sthiti**:

```text
Again, there are four stations:
the four contaminated aggregates,
only in their own ground.
```

Feature extraction:

```text
New count:
    catasraḥ sthitayaḥ
    four stations

Identification:
    catvāraḥ sāsravāḥ skandhāḥ
    the four contaminated aggregates

Restriction:
    svabhūmāv eva kevalam
    only in their own ground
```

The key construction is:

```text
No additional vijñāna-sthiti
    because undesired residence is not counted as station of knowing

But in another sense:
    four sthitayaḥ = four contaminated aggregates
    restricted to their own ground
```

## 6. Literal Translation

```text
There are no others, because residence there is undesired. Again, the four stations are the four contaminated aggregates, only in their own ground.
```

A smoother literal rendering:

```text
No other [knowledge-stations] are admitted, because dwelling there is not desired. Further, there are four stations: the four contaminated aggregates, only within their own ground.
```

---

## 7. Philosophical Translation

```text
The system admits no further knowledge-stations beyond those already named, since those other states are not desirable abodes for established knowing. Yet “station” has another technical sense: the four contaminated aggregates function as stations, each only within its own proper ground.
```

Organon-clean:

```text
The world-field is not expanded arbitrarily. A station of knowing must be a possible site of abiding, not merely any state of suffering or interruption. In a second register, the contaminated aggregates themselves are stations, but only within their proper domain-ground. Knowledge is world-positioned, and its supports are ground-specific.
```

## 8. Technical Vocabulary

### anicchā-avasana

```text
anicchā-avasana
    undesired residence
    unwanted dwelling
    abiding that is not wished for
```

This explains why other states are not counted as **vijñāna-sthiti**. The point is not that they do not exist, but that they are not proper stations of established conditioned knowing.

Working note:

```text
not every state of existence is a knowledge-station
```

### anya

```text
anya
    other
    additional
```

Here **na anye** excludes further members from the seven **vijñāna-sthiti**.

### sthiti

```text
sthiti
    standing
    establishment
    station
```

This verse makes clear that **sthiti** is being used in more than one sense.

```text
vijñāna-sthiti
    station of conditioned knowing

catasraḥ sthitayaḥ
    four aggregate-stations
```

So:

```text
sthiti is not univocal here.
```

### punaḥ

```text
punaḥ
    again
    further
    in another respect
```

Here **punaḥ** marks a second technical register. The verse is not merely repeating the seven knowledge-stations. It introduces another kind of station.

### sāsrava

```text
sa-āsrava
    with outflows
    contaminated
    belonging to conditioned bondage
```

The **sāsrava skandhas** are aggregates still marked by outflow, contamination, or bondage. They are not pure or liberated supports.

### skandha

```text
skandha
    aggregate
    aggregate-configuration
```

Here the four contaminated aggregates are likely the four aggregates other than vijñāna:

```text
rūpa
vedanā
saṃjñā
saṃskāra
```

This is important because **vijñāna** is not itself treated here as its own station. Rather, conditioned knowing is stationed with respect to the other aggregates.

Working rule:

```text
vijñāna is not its own support-station.
It stands in relation to the other contaminated aggregates.
```

### svabhūmi

```text
sva-bhūmi
    own ground
    proper ground
    own level/domain-ground
```

The restriction is emphatic:

```iast
svabhūmāv eva kevalam
```

Meaning:

```text
only in its own ground
exclusively within its proper level
```

This prevents arbitrary cross-domain projection.

### eva kevalam

```text
eva
    only / just

kevalam
    exclusively / only
```

The double emphasis strengthens the restriction:

```text
only, and only there
```

## 9. Logical Determination

VAkK 3.7 gives:

```text
World as ground-specific aggregate-support.
```

The movement from 3.5 to 3.7 is:

```text
body/perception configuration
    ↓
knowledge-station
    ↓
being-abode
    ↓
aggregate-support in its own ground
```

The verse does two things:

```text
1. closes the count of knowledge-stations
2. opens a second sense of station as aggregate-support
```

The logical distinction is:

```text
vijñāna-sthiti
    station of conditioned knowing

sāsrava-skandha-sthiti
    contaminated aggregate as support-station
```

So the Law of Appearance becomes more exact:

```text
Knowing is stationed,
but its supports are aggregates,
and those aggregates function only within their proper ground.
```

## 10. Organon Interpretation

This verse is important for the Semantic Dataset model.

A dataset cannot allow arbitrary feature transfer. A feature has a **proper ground**.

```text
svabhūmi
    proper ground of a determination
```

So in Organon terms:

```text
No determination floats freely.
No aggregate-support operates outside its own ground.
No knowing stands without lawful support.
```

This preserves the marks we have been observing:

```text
The Semantic Dataset carries the Principle.
The Agent carries the Result.
Karma carries the transition.
The aggregate-state provides lawful support for conditioned knowing.
```

The verse also clarifies the wheel:

```text
skandha
    aggregate-state / support configuration

karma
    wraparound transition toward Next()

svabhūmi
    ground constraint of the support
```

So:

```text
Next(skandha)
    is not arbitrary.
    It manifests within ground-specific constraints.
```

This is why Lokanirdeśa is not merely cosmology. It is laying out the lawful appearance-space in which karma can later be intelligible.
