## 5. Syntax / Construction

This verse is a hinge.

First, it closes the preceding discussion of **sthiti**:

```iast
vijñānaṃ na sthitiḥ proktam
```

Meaning:

```text
Vijñāna itself is not declared to be a station.
```

The previous verse identified the four contaminated aggregates as stations in their own ground. This verse clarifies that **vijñāna** is not counted as another such station.

Formal closure:

```text
vijñāna
    is not itself counted as sthiti

because:
    conditioned knowing stands with respect to supports
    but is not itself another support-station in this enumeration
```

Second, the verse opens a new classificatory topic:

```iast
catuṣkoṭi tu saṃgrahe
```

Meaning:

```text
But in the summary/classification, there is a fourfold division.
```

Third, it names that fourfold division:

```iast
catasro yonayas tatra sattvānām aṇḍajādayaḥ
```

Meaning:

```text
There are four yonis of beings, beginning with the egg-born.
```

The formal operation is:

```text
Close:
    vijñāna is not itself sthiti

Open:
    beings are classified by four birth-modes
```

So the verse shifts from:

```text
station of knowing / aggregate support
```

to:

```text
mode of arising / birth-source
```

---

## 6. Literal Translation

```text
Vijñāna is not declared to be a station. But in the classification there is a fourfold division: there are four yonis of beings, beginning with the egg-born.
```

A smoother literal rendering:

```text
Knowledge itself is not taught as a station. But in the summary, the division is fourfold: for beings there are four birth-modes, beginning with the egg-born.
```

---

## 7. Philosophical Translation

```text
Conditioned knowing is not itself counted as a support-station. Having closed that issue, the analysis turns to another dimension of the world-field: sentient beings arise through four generative modes, beginning with egg-birth.
```

Organon-clean:

```text
The world-field is now no longer described only as station and support. It is also a field of instantiation. Beings do not merely stand somewhere; they arise through determinate modes of generation.
```

## 8. Technical Vocabulary

### vijñāna

```text
vijñāna
    knowledge
    conditioned knowing
    determinate cognitive event
```

Here **vijñāna** must not be inflated into Puruṣa, Buddhi, Prajñā, or Absolute Knowledge. It is conditioned knowing within the Abhidharma analysis.

The verse says:

```text
vijñānaṃ na sthitiḥ proktam
```

So:

```text
vijñāna
    is stationed
    but is not itself counted as a station here
```

This protects the distinction between:

```text
vijñāna-sthiti
    station of conditioned knowing

vijñāna-skandha
    the knowing aggregate itself
```

### sthiti

```text
sthiti
    standing
    establishment
    station
```

The previous verses developed several senses of **sthiti**:

```text
vijñāna-sthiti
    knowledge-station

sāsrava-skandha-sthiti
    contaminated aggregate as support-station
```

This verse prevents a false inference:

```text
Because there are vijñāna-sthiti,
it does not follow that vijñāna itself is another sthiti.
```

### prokta

```text
prokta
    said
    taught
    declared
```

The classification is doctrinally controlled. A category is not admitted merely because it seems possible. It must be **prokta**, taught/declared in the system.

### catuṣ-koṭi

```text
catuṣ-koṭi
    fourfold
    four-cornered
    having four divisions/classes
```

Here the term marks a fourfold classificatory division, not the later dialectical catuṣkoṭi of assertion/negation logic.

### saṃgraha

```text
saṃgraha
    summary
    collection
    inclusion
    classificatory gathering
```

In this verse, **saṃgraha** introduces a new classificatory register: the collection of beings according to birth-mode.

### yoni

```text
yoni
    womb
    birth-source
    generative matrix
    mode of arising
```

For this project, **birth-mode** or **generative matrix** is usually better than merely “womb,” because the four yonis include modes that are not womb-birth.

Working rule:

```text
yoni
    not simply biological womb
    but mode of sentient arising
```

### sattva

```text
sattva
    being
    sentient being
    sentient continuity
```

As before, **sattva** must not be substantialized into ātman. It marks a sentient continuity appearing within the world-field.

### aṇḍaja

```text
aṇḍaja
    egg-born
```

The phrase **aṇḍajādayaḥ** means “egg-born and the rest.” The other members of the fourfold set are supplied by the larger doctrinal classification:

```text
aṇḍaja
    egg-born

jarāyuja
    womb-born

saṃsvedaja
    moisture-born

upapāduka
    spontaneously arisen
```

## 9. Logical Determination

VAkK 3.8 gives:

```text
World as birth-mode classification.
```

The logical movement is:

```text
knowledge-station
    ↓
aggregate-support
    ↓
birth-mode
```

This is the beginning of Block C.

Block B established:

```text
where conditioned knowing can stand
and what aggregate-supports function as stations
```

Block C begins by asking:

```text
How do sentient continuities arise into the world-field?
```

The answer begins with **yoni**:

```text
Beings arise through four generative modes.
```

So the world is not merely:

```text
domain
station
support
```

It is also:

```text
generative arising
```

## 10. Organon Interpretation

This verse opens the transition-system of Loka.

For the Semantic Dataset, **yoni** is a new feature-class:

```text
Feature:
    BirthMode

Values:
    EggBorn
    WombBorn
    MoistureBorn
    SpontaneouslyArisen
```

This must be kept at the Loka level. We do not yet unfold full Agential Karma.

```text
In Lokanirdeśa:
    karma is only the required transition-principle / dataset feature

In Karmanirdeśa:
    karma becomes the full agential action-system
```

So for Block C:

```text
Karma is not yet moral catastrophe.
Karma is a Principle-feature that conditions transition.
```

The verse therefore begins the modeling of `Next()` without yet analyzing the whole karmic machinery:

```text
Current sentient continuity
    receives a birth-mode feature
    and arises into a determinate world-position
```

But the system immediately guards against a false self-model:

```text
sattva
    means sentient continuity,
    not ātman.
```

Thus:

```text
No self is being born as a substance.
A sentient continuity is classified by a lawful mode of arising.
```
