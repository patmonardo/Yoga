## 5. Syntax / Construction

This verse gives the definition of **antarābhava**.

First clause:

```iast
mṛtyupapattibhavayor antarā bhavatīha yaḥ
```

Meaning:

```text
That which exists here between death-existence and rebirth-existence ...
```

The key pair is:

```text
mṛtyu-bhava
    death-existence

upapatti-bhava
    rebirth-existence / arising-into-birth existence
```

Between these two there is an interval:

```text
mṛtyu-bhava
    ↓
antarā-bhava
    ↓
upapatti-bhava
```

Second clause:

```iast
gamyadeśānupetatvān nopapanno 'ntarābhavaḥ
```

Meaning:

```text
Because it has not yet reached the destination-place, it is not yet upapanna; it is antarābhava.
```

The definition depends on a negative mark:

```text
not yet arrived at destination-place
    → not yet reborn
    → intermediate existence
```

Formal operation:

```text
If:
    after death-existence
    before rebirth-existence
    destination-place not yet reached
Then:
    antarābhava
```

---

## 6. Literal Translation

```text
That which exists here between death-existence and rebirth-existence, because it has not reached the destination-place, is not yet reborn; it is intermediate existence.
```

A smoother literal rendering:

```text
What exists in the interval between death and rebirth, not yet having reached the destination-place, is not yet arisen there; this is the intermediate existence.
```

---

## 7. Philosophical Translation

```text
The system posits an intermediate phase between death and rebirth. Since the destination of rebirth has not yet been reached, this phase is not itself the new birth. It is the between-state: antarābhava.
```

Organon-clean:

```text
The world-field now requires a transition-state. The sentient continuity is no longer in the previous birth, but it has not yet arrived at the next destination. This interval is not a soul in transit; it is a conditioned transition-phase in the Loka dataset.
```

## 8. Technical Vocabulary

### mṛtyu

```text
mṛtyu
    death
    death-event
```

Here **mṛtyu** marks the limit of a previous aggregate-configuration.

### upapatti

```text
upapatti
    rebirth
    arising-into-birth
    occurrence into a new mode of existence
```

This was already important in VAkK 3.3, where the formless domain was fourfold by **upapatti**. Here it becomes part of the transition problem.

### bhava

```text
bhava
    existence
    becoming
    mode of being-there
```

The compound **mṛtyu-upapatti-bhavayoḥ** indicates two bhava-poles:

```text
mṛtyu-bhava
    death-existence

upapatti-bhava
    rebirth-existence
```

### antarā

```text
antarā
    between
    in the interval
```

This gives the formal structure of the verse. **Antarābhava** is defined by between-ness.

### antarābhava

```text
antarābhava
    intermediate existence
    between-state existence
    transition-phase between death and rebirth
```

Critical restraint:

```text
antarābhava is not a soul-substance.
It is a conditioned transition-phase of sentient continuity.
```

The verse defines it through non-arrival:

```text
not yet reached destination
not yet reborn
therefore intermediate
```

### gamya-deśa

```text
gamya-deśa
    destination-place
    place to be reached
    target domain/location of rebirth
```

This is a key Loka term. Transition is not abstract. It is oriented toward a destination-place.

### anupetatva

```text
anupetatva
    not-having-reached
    non-arrival
```

The intermediate being is defined by this lack:

```text
it has not yet reached the gamya-deśa
```

### upapanna

```text
upapanna
    arisen
    reborn
    arrived into birth
```

The verse says **na upapannaḥ**. The antarābhava is not yet the reborn being in the destination-place.

## 9. Logical Determination

VAkK 3.10 gives:

```text
World as intermediate transition-state.
```

The previous verse gave:

```text
being-class
    → permitted birth-mode
```

This verse now adds the transition interval:

```text
death-existence
    → intermediate existence
    → rebirth-existence
```

The formal determination:

```text
antarābhava
    = transition-state defined by non-arrival at gamya-deśa
```

This is the beginning of the continuity problem:

```text
How can there be transition without ātman?
```

The verse does not yet answer fully. It only posits the between-state needed by the system.
