# VAK_3.93 — Kalpa Machinery: Four Phases and the Mahākalpa

## 1. Sanskrit — Devanāgarī

```sanskrit
विवर्ततेऽथ संवृत्त आस्ते संवर्तते समम् ।
ते ह्यशीतिर्महाकल्पस्तदसंख्यत्रयोद्भवम् ॥ VAkK_3.93 ॥
```

## 2. Sanskrit — IAST

```iast
vivartate 'tha saṃvṛtta āste saṃvartate samam /
te hy aśītir mahākalpaḥ tad asaṃkhya-trayodbhavam // VAkK_3.93 //
```

## 3. Source Caution

The source reading is:

```iast
vivartate 'tha saṃvṛtta āste saṃvartate samam /
te hyaśītirmahākalpaḥ tadasaṃkhyatrayodbhavam // VAkK_3.93 //
```

Sandhi has been lightly separated:

```iast
vivartate atha saṃvṛttaḥ āste saṃvartate samam /
te hi aśītiḥ mahā-kalpaḥ tad-asaṃkhya-traya-udbhavam //
```

The first pāda is read as the four great phases of the kalpa-wheel: the world expands, stands expanded, contracts, and stands contracted. Each phase is equal in duration. The `te` refers to the twenty-kalpa phases: twenty for expansion, twenty for standing expanded, twenty for contraction, and twenty for standing contracted, totaling eighty intermediate kalpas. The second pāda links this mahākalpa to the arising of the three asaṃkhyeyas, a point continued in the next verse on buddhahood.

## 6. Literal Translation

It expands; then, having expanded, it stands; it contracts, equally. Those are indeed eighty: the great kalpa. From that arises the three asaṃkhyeyas.

## 7. Philosophical Translation

The kalpa-wheel is now stated in its complete fourfold form. The world expands, stands in its expanded state, contracts, and stands in its contracted state. Each phase is equal in duration. These four phases, each counted as twenty intermediate kalpas, make eighty intermediate kalpas: this is the mahākalpa. From this scale the three asaṃkhyeyas are reckoned.

## 9. Doctrinal Determination

VAkK_3.90 introduced contraction and expansion kalpas. VAkK_3.91–3.92 gave twenty intermediate kalpas for the standing expanded world. VAkK_3.93 now completes the fourfold kalpa-wheel.

```text
vivarta
    → expansion
    → 20 intermediate kalpas

vivarta-sthāyin / saṃvṛtta āste
    → standing expanded
    → 20 intermediate kalpas

saṃvarta
    → contraction
    → 20 intermediate kalpas

saṃvarta-sthāyin
    → standing contracted
    → 20 intermediate kalpas
```

Thus:

```text
20 + 20 + 20 + 20
    = 80 intermediate kalpas
    = 1 mahākalpa
```

## 10. Logical Determination

This verse gives the full temporal schema of the world-wheel.

```text
formation
    → standing-formed
        → dissolution
            → standing-dissolved
                = one full mahākalpa
```

The wheel turns not merely by motion, but by alternating motion and standing.

## 11. Interpretive Note

The precise lingo matters:

```text
vivarta
    = expansion / formation / unfolding

vivarta-sthāyin
    = standing after expansion

saṃvarta
    = contraction / dissolution / enfolding

saṃvarta-sthāyin
    = standing after contraction
```

The verse itself expresses the standing-expanded phase by `saṃvṛtta āste`: once expanded, it stands. The contracted-standing phase is implied by the equal fourfold total and is normally named `saṃvarta-sthāyin` in the kalpa scheme.

This is the language of a wheel turning with pauses:

```text
turning outward
    → standing outward
        → turning inward
            → standing inward
```

## 12. Organon Interpretation

Light contact point:

```text
The mahākalpa is a four-phase temporal wheel.
```

For this pass, we stay close to the Kośa. The larger world-cycle is not a simple circle but a structured alternation of process and state: expansion-process, expanded-state, contraction-process, contracted-state.
