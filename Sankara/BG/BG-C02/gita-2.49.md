# Analysis of Bhagavad Gītā 2.49

## I. Sanskrit Text Structure

### A. Base Verse
```sanskrit
दूरेण ह्यवरं कर्म बुद्धियोगाद्धनञ्जय ।
बुद्धौ शरणमन्विच्छ कृपणाः फलहेतवः ॥ ४९ ॥
```

### B. Key Technical Terms
1. बुद्धियोग (buddhiyoga)
   - Reason-synthesis
   - Pure understanding
   - Synthetic unity

2. अवर कर्म (avara karma)
   - Lower action
   - Mere doing
   - Without understanding

3. कृपण (kṛpaṇa)
   - Miserable/pitiable
   - Result-seekers
   - Missing truth

# Analysis of Śaṅkara's Commentary on BG 2.49

## I. Opening Framework

### A. Initial Context
```sanskrit
यत्पुनः समत्वबुद्धियुक्तमीश्वराराधनार्थं कर्मोक्तम् , एतस्मात्कर्मणः
```

1. Higher Action
   - समत्वबुद्धियुक्त = united with equanimity
   - ईश्वराराधनार्थ = for worship of Ishvara
   - Pure synthetic unity

2. As Standard
   - For comparison
   - Clear distinction
   - True understanding

## II. Analysis of Lower Action

### A. Technical Description
```sanskrit
दूरेण अतिविप्रकर्षेण अत्यन्तमेव हि अवरम् अधमं निकृष्टं कर्म फलार्थिना क्रियमाणं
```

1. Triple Emphasis
   - दूरेण = far
   - अतिविप्रकर्षेण = extremely distant
   - अत्यन्तम् = completely

2. Nature Analysis
   - अवर = lower
   - अधम = inferior
   - निकृष्ट = base

### B. Result Problem
```sanskrit
जन्ममरणादिहेतुत्वात्
```
   - Birth-death causation
   - Bondage creation
   - Continued existence

## III. Two Types of Buddhi

### A. Higher Understanding
```sanskrit
योगविषयायां बुद्धौ तत्परिपाकजायां वा साङ्‍ख्यबुद्धौ
```

1. Yoga Buddhi
   - योगविषया = yoga-oriented
   - Pure understanding
   - Dynamic stability

2. Sāṅkhya Buddhi
   - परिपाकजा = born of maturity
   - Complete understanding
   - Original insight

### B. Refuge Nature
```sanskrit
शरणम् आश्रयमभयप्राप्तिकारणम्
```

1. Technical Terms
   - शरण = refuge
   - आश्रय = shelter
   - अभयप्राप्ति = fearlessness attainment

2. Final Goal
   - परमार्थज्ञानशरण
   - Ultimate knowledge refuge
   - Pure presence

## IV. Analysis of Result-Seekers

### A. Technical Description
```sanskrit
कृपणाः दीनाः फलहेतवः फलतृष्णाप्रयुक्ताः सन्तः
```

1. Nature Analysis
   - कृपण = miserable
   - दीन = wretched
   - फलतृष्णाप्रयुक्त = impelled by desire

2. Scriptural Support
```sanskrit
यो वा एतदक्षरं गार्ग्यविदित्वास्माल्लोकात्प्रैति स कृपणः
```
   - Bṛhadāraṇyaka reference
   - Missing imperishable
   - True poverty

## V. Philosophical Framework

### A. Action Levels
1. Higher Action
   - With equanimity
   - For Ishvara
   - Pure synthesis

2. Lower Action
   - Result-driven
   - Birth-death cause
   - Missing truth

### B. Understanding Types
1. Yoga Buddhi
   - Pure understanding
   - Dynamic stability
   - Original presence

2. Sāṅkhya Buddhi
   - Complete knowledge
   - Ultimate truth
   - Pure insight

## VI. Technical Implications

### A. Progressive Path
1. Beyond Lower Action
   - Result-driven
   - Birth-death cycle
   - Missing truth

2. Through Understanding
   - As refuge
   - Pure presence
   - Original freedom

### B. Final Resolution
1. Ultimate Knowledge
   - Pure understanding
   - Complete shelter
   - Original stability

2. True Freedom
   - Beyond results
