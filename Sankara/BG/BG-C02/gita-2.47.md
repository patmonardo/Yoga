# Analysis of Bhagavad Gītā 2.47

## I. Sanskrit Text Structure

### A. Base Verse
```sanskrit
कर्मण्येवाधिकारस्ते मा फलेषु कदाचन ।
मा कर्मफलहेतुर्भूर्मा ते सङ्गोऽस्त्वकर्मणि ॥ ४७ ॥
```

### B. Key Technical Terms
1. अधिकार (adhikāra)
   - Qualification/right
   - Domain of action
   - Practical sphere

2. कर्मफलहेतु (karmaphalahetū)
   - Cause of action-results
   - Agency structure
   - Result orientation

3. सङ्ग (saṅga)
   - Attachment
   - False identification
   - Wrong relation

4. अकर्मण् (akarman)
   - Non-action
   - Action transcendence
   - Pure presence

# Analysis of Śaṅkara's Commentary on BG 2.47

## I. Initial Framework

### A. Domain Limitation
```sanskrit
कर्मण्येव अधिकारः न ज्ञाननिष्ठायां ते तव
```

1. Technical Distinction
   - कर्मणि = in action
   - Not in ज्ञाननिष्ठा
   - Clear separation

2. Qualification Structure
   - अधिकार = right/domain
   - Proper sphere
   - Limited scope

## II. Analysis of Desire

### A. Result Attachment
```sanskrit
तत्र च कर्म कुर्वतः मा फलेषु अधिकारः अस्तु, कर्मफलतृष्णा मा भूत् कदाचन कस्याञ्चिदप्यवस्थायाम्
```

1. Triple Analysis
   - कर्म कुर्वत् = while doing action
   - फलेषु अधिकार = right to results
   - कर्मफलतृष्णा = thirst for results

2. Complete Negation
   - कदाचन = at any time
   - कस्याञ्चिदप्यवस्थायाम् = in any state
   - Absolute prohibition

### B. Causation Problem
```sanskrit
यदा कर्मफले तृष्णा ते स्यात् तदा कर्मफलप्राप्तेः हेतुः स्याः
```

1. Causal Chain
   - तृष्णा leads to
   - Being हेतु
   - Result attainment

2. Technical Structure
   - Desire as cause
   - Agency problem
   - Bondage creation

## III. Birth-Action Analysis

### A. Result Orientation
```sanskrit
यदा हि कर्मफलतृष्णाप्रयुक्तः कर्मणि प्रवर्तते तदा कर्मफलस्यैव जन्मनो हेतुर्भवेत्
```

1. Action Nature
   - तृष्णाप्रयुक्त = impelled by desire
   - प्रवर्तते = proceeds
   - Wrong motivation

2. Birth Result
   - जन्मन् as फल
   - Bondage cycle
   - Continued existence

### B. Non-Action Problem
```sanskrit
यदि कर्मफलं नेष्यते, किं कर्मणा दुःखरूपेण
```

1. False Logic
   - If no result desired
   - Why painful action?
   - Wrong understanding

2. Resolution Need
   - Beyond result/non-result
   - Pure activity
   - Original presence

## IV. Final Warning

### A. Attachment to Non-Action
```sanskrit
मा ते तव सङ्गः अस्तु अकर्मणि अकरणे प्रीतिर्मा भूत्
```

1. Technical Terms
   - सङ्ग = attachment
   - अकर्मण् = non-action
   - प्रीति = love/preference

2. Complete Negation
   - Of inaction attachment
   - False resolution
   - Wrong understanding

## V. Philosophical Framework

### A. Action Structure
1. Pure Domain
   - Limited to karma
   - Not in knowledge
   - Clear distinction

2. Without Desire
   - No result-right
   - No thirst
   - Pure activity

### B. Resolution Path
1. Beyond Agency
   - Not the cause
   - Beyond desire
   - Original activity

2. Beyond Opposition
   - Not through non-action
   - Pure presence
   - Original unity

   - Through pure action
